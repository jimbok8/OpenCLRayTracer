DESCRIPTION

Class Scene - Contains operation for loading a 3D scene from file, formatting the data and uploading to GPU memory

The scene file that can be loaded by this class is a text file that contains description of an object at every line. Supported types of objects are: SCENE,LIGHT, and MESH. Light example:

LIGHT 0.0 10.0 -10.0 1000.0

Light above is located at x-y-z position (first 3 floats) and the light intensity is the last number

Sphere example:

SPHERE 0.0 0.0 50.0 5.0

Sphere above is located at x-y-z position, and has radius set by the fourth number

Mesh example:

MESH models\swfighter\star-wars-vader-tie-fighter.obj

Mesh above is loaded from file set by the path, that follows the word MESH, relative to working directory. The mesh must be in Wavefront OBJ format, and the material file also should be located at the indicated path.
